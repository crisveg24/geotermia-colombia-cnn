# Scripts Directory

Scripts principales para ejecutar el proyecto.

## Orden de ejecución

1. `download_dataset.py`: Descargar imágenes de Google Earth Engine
2. `prepare_dataset.py`: Preparar y procesar imágenes para CNN
3. `train_model.py`: Entrenar el modelo CNN
4. `evaluate_model.py`: Evaluar métricas del modelo
5. `visualize_results.py`: Generar gráficos y visualizaciones
6. `predict.py`: Hacer predicciones en nuevas imágenes
